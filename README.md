# ANSI TERM

This library is meant to implement a majority of the popular-use ansi-terminal escape codes.

Currently Supported:
- [x] Colors
    - [x] ISO 4bit colors
    - [x] 8bit 256 colors
    - [x] 24bit true RGB colors

Planned Support:
- [ ] Cursor Controls
- [ ] Erase functions
- [ ] Screen Modes
- [ ] Single character functions
